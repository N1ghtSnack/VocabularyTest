\# 📚 会计词汇智能测验系统



> \*\*项目说明\*\*：本系统代码完全由 AI 生成，是一个纯前端、零依赖的词汇测验工具。示例词汇表及会计定义来源于清华大学经济管理学院 \*\*许达（Da Xu）\*\* 老师开设的 \*\*《会计学原理》\*\* 课程课件（Class 1–9），涵盖会计基本概念、财务报表、交易分析、调整分录、存货、固定资产、负债等核心术语，并补充了专业的会计定义。



\## ✨ 主要功能



\- \*\*智能计时评估\*\*  

&#x20; 每道题 10 秒倒计时：

&#x20; - 10 秒内正确 → \*\*掌握\*\*

&#x20; - 超时后正确 → \*\*不熟\*\*

&#x20; - 错误/跳过 → \*\*完全不会\*\*

&#x20; - 点击「释义」按钮 → \*\*未掌握释义\*\*



\- \*\*细分掌握类别\*\*  

&#x20; 实时统计各类词汇数量，学习报告清晰展示薄弱环节。



\- \*\*灵活的学习流程\*\*

&#x20; - 全新开始（随机顺序）

&#x20; - 仅复习弱项（完全不会 + 未掌握释义 + 尚未判断）

&#x20; - 暂停/继续（失去焦点自动暂停，当前词移至末尾）



\- \*\*进度持久化\*\*

&#x20; - 一键保存当前进度为 JSON 文件

&#x20; - 加载之前保存的进度文件，无缝继续



\- \*\*完全自定义词汇表\*\*  

&#x20; 使用 CSV 文件（UTF-8 编码）维护词汇，每行包含：`英文,中文,定义`。  

&#x20; 内置会计专业定义，也可扩展为其他学科。



\- \*\*移动端优化\*\*  

&#x20; 答题界面无需横滑滚动，按钮适配触摸操作。



\## 🚀 快速开始



\### 1. 下载或克隆仓库

```bash

git clone https://github.com/你的用户名/vocabulary-test.git

cd vocabulary-test

```



\### 2. 准备词汇表

\- 编辑 `vocab.csv` 文件（第一行为表头：`英文,中文,定义`）

\- 每行一个词汇，例如：

&#x20; ```csv

&#x20; asset,资产,企业拥有或控制的、预期能带来经济利益的资源。

&#x20; liability,负债,企业过去的交易形成的现时义务，预期会导致经济利益流出。

&#x20; ```



\### 3. 打开测验页面

\- \*\*本地测试\*\*：使用 Chrome 等浏览器直接打开 `index.html`；若自动加载 CSV 失败，手动点击「选择 CSV 词汇表」上传文件。

\- \*\*部署到服务器\*\*：将所有文件上传至任意静态 Web 服务器即可。



\## 🖥️ 部署到 GitHub Pages



1\. 在 GitHub 上创建新仓库，命名为 `vocabulary-test`（公开）。

3\. 进入仓库 \*\*Settings → Pages\*\*，将 Branch 设为 `main`，文件夹为 `/ (root)`，保存。

4\. 等待 1-2 分钟，访问 `https://你的用户名.github.io/vocabulary-test/`。

\## 🎮 使用指南



\### 菜单界面

\- 「全新开始」：加载词汇表后随机顺序开始测验。

\- 「选择 CSV 词汇表」：手动上传 CSV 文件（当自动加载失败时使用）。

\- 「加载进度文件」：恢复之前保存的 `.json` 进度文件。



\### 测验界面

\- 在倒计时 10 秒内选择正确翻译 → \*\*掌握\*\*

\- 倒计时结束后选择正确翻译 → \*\*不熟\*\*

\- 选择错误或点击「跳过」→ \*\*完全不会\*\*

\- 点击「释义」查看详细定义，同时该词记为 \*\*未掌握释义\*\*

\- 点击「暂停」或离开页面（失去焦点）→ 当前词汇移至队列末尾，计时重置



\### 学习报告界面

\- 分类展示已掌握、不熟、完全不会、未掌握释义及尚未判断的词汇。

\- 点击任意单词可弹出详细定义。

\- 「复习弱项」：仅对“完全不会 + 未掌握释义 + 尚未判断”的词汇重新测验。

\- 「暂时退出并保存进度文件」：导出当前进度 JSON 文件，并停留在报告界面。



\## 🛠️ 自定义与扩展



\### 修改词汇表

直接编辑 `vocab.csv` 即可，支持任意学科。表头必须为 `英文,中文,定义`。



\### 修改计时长度

在 `index.html` 的 JavaScript 中找到 `timeLeft = 10`，改为您需要的秒数。



\### 修改选项数量

调整 `getRandomDistractors` 函数中的 `count=3` 参数。
