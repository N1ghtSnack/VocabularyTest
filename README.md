# 📚 会计词汇智能测验系统

> **代码完全由 AI 生成**，是一个纯前端、零依赖的词汇测验工具。  
> 示例词汇表及会计定义来源于 **清华大学经济管理学院** [许达（Da Xu）] 老师开设的 **《会计学原理》** 课程（Class 1–9 课件），涵盖会计基本概念、财务报表、交易分析、调整分录、存货、固定资产、负债等核心术语。

## ✨ 主要功能

- **智能计时评估**  
  每道题 10 秒倒计时：
  - 10 秒内正确 → **掌握**
  - 超时后正确 → **不熟**
  - 错误/跳过 → **完全不会**
  - 点击「释义」按钮 → **未掌握释义**

- **细分掌握类别**  
  实时统计各类词汇数量，学习报告清晰展示薄弱环节。

- **灵活的学习流程**
  - 全新开始（随机顺序）
  - 仅复习弱项（完全不会 + 未掌握释义 + 尚未判断）
  - 暂停/继续（失去焦点自动暂停，当前词移至末尾）

- **进度持久化**
  - 一键保存当前进度为 JSON 文件
  - 加载之前保存的进度文件，无缝继续

- **完全自定义词汇表**  
  使用 CSV 文件（UTF-8 编码）维护词汇，每行包含：`英文,中文,定义`。  
  内置会计专业定义，也可扩展为其他学科。

- **移动端优化**  
  答题界面无需横滑滚动，按钮适配触摸操作。

## 🎮 使用指南

### 菜单界面
- **「全新开始」**：加载词汇表后随机顺序开始测验。
- **「选择 CSV 词汇表」**：手动上传 CSV 文件（当自动加载失败时使用）。
- **「加载进度文件」**：恢复之前保存的 `.json` 进度文件。

### 测验界面
- 在倒计时 10 秒内选择正确翻译 → **掌握**
- 倒计时结束后选择正确翻译 → **不熟**
- 选择错误或点击「跳过」→ **完全不会**
- 点击「释义」查看详细定义，同时该词记为 **未掌握释义**
- 点击「暂停」或离开页面（失去焦点）→ 当前词汇移至队列末尾，计时重置

### 学习报告界面
- 分类展示已掌握、不熟、完全不会、未掌握释义及尚未判断的词汇。
- 点击任意单词可弹出详细定义。
- **「复习弱项」**：仅对“完全不会 + 未掌握释义 + 尚未判断”的词汇重新测验。
- **「暂时退出并保存进度文件」**：导出当前进度 JSON 文件，并停留在报告界面。

## 🛠️ 自定义与扩展

### 修改词汇表
直接编辑 `vocab.csv` 即可，支持任意学科。表头必须为 `英文,中文,定义`。

### 修改计时长度
在 `index.html` 的 JavaScript 中找到 `timeLeft = 10`，改为您需要的秒数。

### 修改选项数量
调整 `getRandomDistractors` 函数中的 `count=3` 参数。
